Atividade: Desenvolvimento de um Site — Top 3 Cidades dos Sonhos

Nesta atividade, o objetivo é desenvolver um site apresentando as três cidades escolhidas de acordo com o gosto pessoal de cada aluno. O site deve destacar os principais motivos que tornam esses destinos especiais, considerando aspectos como cultura, turismo, lazer, qualidade de vida, gastronomia, arquitetura ou experiências que cada cidade pode oferecer.
